# Dotfiles for macOS

```shell
make init
make deploy
make git
make emacs
make install
```

## Additional tools

``` shell
make help
```

## Update

``` shell
make update
```

## Install one by one

``` shell
./install ruby
```

Install libraries.

``` shell
./install r ruby
```

Install language and libraries.

``` shell
./install t ruby
```

## Uninstall

``` shell
./uninstall ruby
```
